# 👥 使用者角色（Personas）

## 這是什麼？

這個資料夾存放所有使用者角色的定義。每個角色代表一種使用系統的人。

## 角色怎麼產生？

1. **自動識別**：當 AI 在轉譯需求時，會自動識別出涉及的角色。如果這個角色還沒有定義檔案，AI 會自動建立一個。
2. **手動新增**：你也可以用 `_template.md` 模板手動建立角色。
3. **持續更新**：隨著需求的累積，角色的描述會越來越完整。

## 角色的用途

- 幫助 AI 從不同角度思考需求
- 在衝突偵測時，識別不同角色之間的利益衝突
- 確保每個功能都考慮到所有相關使用者

## 檔案結構

- `_template.md` — 新角色的模板
- `examples/` — 範例角色，可作為參考
- 其他 `.md` 檔案 — 專案中實際定義的角色
