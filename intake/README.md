# 📥 怎麼提交需求？

## 超簡單，三件事就好

1. **你是誰？** 告訴我們你的名字和角色（例如：業務部小王、客服組阿美）
2. **你想要什麼？** 用你自己的話說就好，不用擔心格式
3. **為什麼需要？** 這對你的工作有什麼幫助？

## 你可以怎麼提交

### 方法一：直接跟 AI 說（最簡單）
在 Claude Code 中輸入 `/intake`，AI 會引導你完成

### 方法二：自己寫一份放進來
1. 到 `templates/` 資料夾，挑一個最像你情況的模板
2. 複製一份，填好你的內容
3. 放到 `raw/` 資料夾裡

## 有哪些模板可以用？

| 模板 | 適合什麼情況 |
|------|-------------|
| `quick-idea.md` | 我有一個簡單的想法 |
| `meeting-notes.md` | 會議中討論到的需求 |
| `user-feedback.md` | 使用者回饋或抱怨 |
| `bug-report.md` | 系統有問題要回報 |

## 常見問題

**Q：我不知道怎麼描述，很怕說不清楚？**
A：怎麼說都可以！你平常跟同事怎麼講，就怎麼寫。AI 會幫你整理的。

**Q：可以用口語嗎？**
A：當然可以！「那個東西按下去之後跑很慢」這樣的描述完全 OK。

**Q：我可以補充修改嗎？**
A：隨時都可以！需求變更是正常的，我們的流程就是為此設計的。
